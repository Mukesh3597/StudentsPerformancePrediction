# Students Performance Prediction

## Project Overview
This project predicts the **Reading Score** of students based on demographic and test data using **Linear Regression**.  

## Dataset
- Source: `StudentsPerformance.csv`
- Columns: gender, race/ethnicity, parental level of education, lunch, test preparation course, math score, reading score, writing score

## Files
- `notebooks/1_eda_and_preprocessing.ipynb` → Exploratory Data Analysis + Feature Encoding
- `notebooks/2_model_training.ipynb` → Model Training + Evaluation (MAE, MSE)
- `scripts/predict_user_input.py` → Predict reading score using user input
